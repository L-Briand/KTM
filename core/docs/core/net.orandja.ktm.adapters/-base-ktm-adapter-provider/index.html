BaseKtmAdapterProvider

open class BaseKtmAdapterProvider : KtmAdapter.Provider

BaseKtmAdapter is an open class that implements the KtmAdapter.Provider interface. It provides methods to create adapters for different types and retrieve adapters for a given type.

You can extend it to create your own or use the make method to construct another provider on top of the current one.
