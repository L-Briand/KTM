Section

data class Section(val name: Array<String>, val inverted: Boolean, val parts: Array<MDocument>) : MDocument

A section of a mustache document: '{{# section }} content {{/ section }}'

Can also be an inverted section: '{{^ section }} content {{/ section }}'

Parameters

name

parts of the section name tag. Empty array is illegal, a section cannot be unnamed.

inverted

true if the section is inverted

parts

tokens to renders in order inside this section.
